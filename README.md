# University-OOP-Projects
A collection of Java projects focused on **Object-Oriented Programming (OOP)** principles developed during my Computer Engineering studies.

## Projects

### Coordinate System Simulation
A modular Java application that simulates geometric shapes on a 2D plane. It handles interactions between points, lines, circles, and quadrilaterals.

* **Key Features:** Euclidean distance calculations, area/perimeter logic for shapes.
* **Concepts Used:** Encapsulation (Private fields/Getters & Setters), Composition (Objects as class members).
* **Key Files:** `Point.java`, `Circle.java`, `Line.java`, `Quad.java`, and `coordinateProject.java` (Main).

## Tech Stack
* **Language:** Java 
* **IDE:** NetBeans
